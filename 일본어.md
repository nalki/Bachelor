### ある
| 구분 | 반말 (보통체) | 존댓말 (정중체) |
| :--- | :--- | :--- |
| 현재긍정 | 本がある | 本があります |
| 현재부정 | 本がない | 本がないです / ありません |
| 과거긍정 | 本があった | 本がありました |
| 과거부정 | 本가なかった | 本がなかったです  / ありませんでした |

### いる
| 구분 | 반말 (보통체) | 존댓말 (정중체) |
| :--- | :--- | :--- |
| 현재긍정 | 猫がいる | 猫がいます |
| 현재부정 | 猫がいない | 猫がいないです  / いません |
| 과거긍정 | 猫がいた | 猫がいました |
| 과거부정 | 猫がいなかった | 猫がいなかったです  / いませんでした |

### い / な형용사 예외 

⭐い로 끝나지만 '나형용사'인 단어들

- 가장 주의해야 할 단어들입니다. '이'로 끝나서 이형용사처럼 보이지만, 실제로는 **な**로 연결하고 활용도 나형용사의 규칙을 따릅니다.

    - きらい: 싫어하다
    - とくい: 자신 있다, 잘하다
    - にがて: 자신 없다, 서툴다
    - さいわい: 다행이다, 행복하다
    - ていねい: 정중하다, 친절하다

⭐な형용사처럼 보이지만 'い형용사'인 경우

- 이 경우는 단어의 끝이 '이'가 아닌 것처럼 느껴지거나, 한자어라서 나형용사일 것 같지만 실제로는 이형용사인 경우입니다.
    - きいろい: 노랗다
    - ちゃいろい: 갈색이다
    - しかくい: 네모나다
    - まるい: 둥글다

### 반대되는 형용사

- 크기·수치 (Size & Value)
    - 높다 / 낮다: 高(たか)い / 低(ひく)い
    - 길다 / 짧다: 長(なが)い / 短(みじか)い
    - 많다 / 적다: 多(おお)い / 少(すく)ない
    - 크다 / 작다: 大(おお)きい / 小(ちい)さい

- 거리·속도 (Distance & Speed)


멀다 / 가깝다: 遠(とお)い / 近(ちか)い 

빠르다 / 느리다: 早(はや)い / 遅(おそ)い 

넓다 / 좁다: 広(ひろ)い / 狭(せま)い 

깊다 / 얕다: 深(ふか)い / 浅(あさ)い 

- 무게·두께 (Weight & Thickness)
무겁다 / 가볍다: 重(おも)い / 軽(かる)い 

두껍다 / 얇다: 厚(あつ)い / 薄(うす)い 

굵다 / 가늘다: 太(ふと)い / 細(ほそ)い 

- 난이도·강도 (Difficulty & Strength)

보안 엔지니어 교육생으로서 학습이나 과제 강도를 말할 때 쓰기 좋습니다.

어렵다 / 쉽다: 難(むずか)しい / 易(やさ)しい 

강하다 / 약하다: 強(つよ)い / 弱(よわ)い 

딱딱하다 / 부드럽다: 硬(かた)い / 柔(やわ)らかい 

- 가격·기분 (Cost & Emotion)
 
비싸다 / 싸다: 高(たか)い / 安(やす)い 

기쁘다 / 슬프다: 嬉(うれ)しい / 悲(かな)しい

좋다 / 나쁘다: 良(よ)い / 悪(わる)い

### 동사의 반대말

- 전자기기 및 불빛 (켜다 / 끄다)

つける: (불을) 켜다, (스위치를) 올리다

けす: (불을) 끄다, 지우다

- 일상 동작 (열다, 넣다, 입다 등)

긍정 의미	    부정 의미
あける 열다	    しめる 닫다
いれる 넣다	    だす 꺼내다
のる 타다	    おりる 내리다
ぬぐ 벗다	    きる 입다 (상의 등)
かう 사다	    うる 팔다

- 이동 및 방향 관련
 
いく (이쿠) : 가다 <-> くる (쿠루) : 오다

はじまる (하지마루) : 시작되다 <-> おわる (오와루) : 끝나다

あがる (아가루) : 올라가다 <-> さがる (사가루) : 내려가다

- 추상적/상태 관련
 
わすれる (와스레루) : 잊다 <-> おぼえる (오보에루) : 외우다/기억하다

わらう (와라우) : 웃다 <-> なく (나쿠) : 울다

ふえる (후에루) : 늘다 <-> へる (헤루) : 줄다
